Usando links externos

O link externo é um link que vai apontar para outro site, ou seja que não é nosso.


Você pode acessar o meu repositório público no GitHub

Você também pode acessar o meu canal no Youtube

Atenção: Sempre que estiver utilizando um link externo o recomentado é utilizar o target, como esta sendo utilizado acima!
Estes parametros relatadados acima faz com que ao clicar no link, o site externo será aberto em outra janela do navegador. Assim não estará se sobrepondo ao seu site.
